# 🤝 Contributing to Levelify

Thank you for your interest in contributing to Levelify!

We welcome contributions from everyone.

## 🚀 Getting Started

1. Fork this repository.
2. Clone your fork.

```bash
git clone https://github.com/YOUR_USERNAME/Levelify.git
```

3. Install dependencies.

```bash
npm install
```

4. Start the project.

```bash
npm run dev
```

---

## 📝 How to Contribute

- Fix bugs
- Improve documentation
- Improve UI
- Add new features
- Improve performance

---

## 📌 Before submitting a Pull Request

- Create a new branch

```bash
git checkout -b feature-name
```

- Commit changes

```bash
git commit -m "Add feature"
```

- Push

```bash
git push origin feature-name
```

- Open a Pull Request.

---

## 📚 Coding Standards

- Write clean code
- Use meaningful variable names
- Keep components reusable
- Follow existing project structure
